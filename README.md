# Luxe Nails & Spa Website

A responsive, accessible website for Luxe Nails & Spa - a premium nail salon business.

## Project Structure
luxe-nails-spa/
│
├── index.html # Homepage
├── services.html # Services and pricing
├── gallery.html # Nail art gallery
├── booking.html # Online booking system
├── about.html # About us page
├── contact.html # Contact information
│
├── css/
│ ├── style.css # Main stylesheet
│ └── reset.css # CSS reset
│
├── js/
│ ├── main.js # Main JavaScript functionality
│ ├── booking.js # Booking system
│ └── gallery.js # Gallery interactions
│
├── images/ # Image assets
├── robots.txt # SEO robots file
├── sitemap.xml # SEO sitemap
└── README.md # Project documentation

## Features

### Part 1: Foundation
- Semantic HTML5 structure with proper accessibility
- ARIA labels and roles for screen readers
- Skip navigation links
- Organized file structure
- SEO-optimized meta tags and structured data

### Part 2: Visual Design
- Responsive CSS Grid and Flexbox layouts
- CSS custom properties for consistent theming
- Mobile-first responsive design
- Interactive elements with smooth transitions
- Typography hierarchy and scale
- Accessible color contrast

### Part 3: Functionality & SEO
- Multi-step booking system with form validation
- Interactive gallery with filtering and lightbox
- JavaScript form validation with error handling
- SEO optimization (meta tags, structured data, sitemap)
- Accessible forms with proper labels
- robots.txt and sitemap.xml
